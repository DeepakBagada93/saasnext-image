# **App Name**: InstaGenius

## Core Features:

- Post Idea Input: User interface to input social media post ideas.
- Style Selection: A dropdown menu allows users to select a specific image style like 'Bold Minimalist'.
- Image Generation: Generative AI tool creates an image based on the user's post idea and the selected 'Bold Minimalist' style.
- Image Display: Display generated images in a visually appealing manner.
- Download Options: Options for users to download generated images in various formats.

## Style Guidelines:

- Primary color: Saturated teal (#4db6ac) for a modern and engaging feel.
- Background color: Light, desaturated teal (#e0f2f1).
- Accent color: Analogous green (#a5d6a7), slightly brighter than the background, for interactive elements.
- Headline font: 'Poppins', sans-serif. Body font: 'PT Sans', sans-serif.
- Use clean and simple icons to represent different functionalities.
- A grid-based layout that ensures content is well-organized and responsive across devices.
- Subtle transition effects to improve user experience.
